# 🐱 ねこカフェ for Engineers

猫の手も借りたいほど忙しいエンジニアのための、ブラウザで動くリラクゼーションアプリです。

## 機能

### 🐾 猫とふれあい
- CSSアニメーションの猫がゆらゆら待っています
- クリックして撫でるとハートが飛び、猫が喜びます
- ランダムな猫メッセージで癒されます

### ⏱️ ポモドーロタイマー
- 25分作業 → 5分休憩のサイクル（時間はカスタマイズ可能）
- 休憩時間になると猫が活発に動き出します
- プログレスバーで残り時間を視覚的に確認

### 🎵 環境音ミキサー
- 🌧️ 雨音
- 🔥 焚き火
- 😺 猫のゴロゴロ音

3つの音をスライダーで自由にミックスできます。Web Audio API で生成しているため、音声ファイル不要でオフラインでも動作します。

### 💬 癒しの名言
- 猫の哲学、偉人の言葉、プログラマーの知恵など60種類を収録
- ボタンを押すたびにランダムで表示

## 使い方

`index.html` をブラウザで開くだけで動きます。

- インストール不要
- 外部依存なし
- サーバー不要
- Mac / Windows / Linux 対応

## ファイル構成

```
index.html   … アプリ本体
style.css    … スタイル・アニメーション
app.js       … タイマー・環境音・名言のロジック
```

## デプロイ

GitHub Pages で公開できます。

1. リポジトリの **Settings → Pages → Build and deployment**
2. Source を **GitHub Actions** に設定
3. `main` ブランチに push すると自動デプロイされます
